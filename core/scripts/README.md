# Core Scripts (boot / utilidades)

Scripts utilitarios genericos do boilerplate. Roda em ambientes Unix-like
(Linux, macOS, WSL). No Windows nativo, use PowerShell equivalentes ou rode
via WSL.

## Scripts planejados

- `bootstrap.sh` - prepara ambiente local de desenvolvimento (instala deps,
  baixa Playwright browsers, cria `.env` a partir de `.env.example`,
  validates).
- `spawn-agent.sh` - dispara um agente a partir do CLI / shell, passando
  request packet via stdin (nunca via args).
